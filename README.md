# Shop
一个iOS端的商城项目，实现了简单的购物流程以及商家的管理功能。数据库为Sqlite，本地数据库。                                                           
这是一个非常好的练习项目，通过此项目可以学习到很多关于iOS开发的基础控件的使用以及逻辑问题处理。                                                  
同时，这也是我第一个完整的iOS项目，并且用这个项目是我的毕设，用时一个半月。
